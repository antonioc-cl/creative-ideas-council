# Creative Ideas Council Agent Skill

Expert experiences council with 6 advisors for events, experiences, space activation, and idea generation.

## Advisors

- **Joe Pine** - Experience economy, perceived value
- **Priya Parker** - Gathering design, purpose
- **Danny Meyer** - Hospitality, service
- **Rory Sutherland** - Counterintuitive ideas
- **Seth Godin** - Organic marketing, tribes
- **Nick Gray** - Repeatable, low-cost events

## Installation

### npx skills add (recommended)
```bash
npx skills add antonioc-cl/creative-ideas-council
```

### Claude Code
```bash
/plugin add https://github.com/antonioc-cl/creative-ideas-council
```

### Claude.ai
Upload the `creative-ideas-council` folder or add via skills panel.

## Usage

Activate when you need help with:

- Event ideas for physical space
- Experience design and activation
- Community building strategies
- Content creation for events
- Space monetization
- Quick executable ideas with low budget

### Example Prompts

```
Use creative-ideas-council to come up with event ideas
Use creative-ideas-council to activate my physical space
Use creative-ideas-council to build community around my venue
Use creative-ideas-council to create memorable experiences
```
